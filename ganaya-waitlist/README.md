# GanáYa — Waitlist

Landing page de waitlist para GanáYa.

## Setup

1. Clonar el repo
2. Crear `.env.local` con las variables de Supabase:

```
NEXT_PUBLIC_SUPABASE_URL=https://tpfubpkfajoyfygdqfpv.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_publishable_key
```

3. `npm install`
4. `npm run dev`
